Entrainement du modèle

Ce "lien" est en fait une fonction mathématique très complexe qu'on appelle modèle. Il en existe de nombreux types (Réseau de neurones, arbres de décision, ...) mais pour simplifier ils peuvent être représentés par une liste de paramètres à déterminer.

Entraîner un modèle, c'est régler ces paramètres. Pour cela on utilise des algorithmes spécialisés (backpropagation) à partir des données du passé.



NB : Cet entrainement peut être très long (plusieurs heures / jours) si on utilise beaucoup de données mais il suffit de le faire une fois.
